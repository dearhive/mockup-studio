Introduction

Welcome to Mockup Studio, a WebGL 3D application for building mockups.

Mockup Studio can be used to create mockups for trifolds, bifolds, stationeries, books, magazines, App UI, Mobile other devices and so on. This current version , is tailored for Trifold, Bifold, letterhead, Flyer, Business Card and UI screens.

Mockup Studio is well suited for designers and studios who benefit from presentation of their creative designs but are limited by traditional PSD mockups and their limited angles/presets.

It is cross-platform, and runs on Google Chrome. For best results use it as an Addon.

 

Key-Features

  • Mockups for nearly any size.
  • Runs on Web Browser, no Photoshop!!
  • Cross Platform. Runs on Linux/Win/Mac.
  • No troublesome Slices anymore..
  • Unlimited presets shared online.
  • Create close-ups like never before... :D
  • Google drive access for shared folders.

Documentation

Installation

To run the Mockup tool/software on Google Chrome Browser user has to:

  • Get Google Chrome; if you don't have it already.
  • Extract the purchased Mockup Studio Pack and locate the "MockupStudio.crx" file.
  • Launch Google Chrome; if you are not using it to view this documentation.
  • Open Google Chrome Extensions Page. Alternatively you can type in the addressbar: chrome://extensions/
  • Drag and Drop the"MockupStudio.crx" file into Extensions Page Hint: A drop to install box will appear on the page..
  • Then the tool can be accessed through from the Icon on the top right. Or by pressing "CTRL + M"

User Interface

When starting the Mockup Studio, similar screen appears.

Selected object

Selected object is the brochure in the screen with yellow lined box around it.

Most Commonly Used Elements:

  • On the center is the scene area where you can create and interact with the brochure and background.
  • On the left are some presets that you can click and the selected brochure will change to the preset. Below that is a small display that shows the FPS indicating the speed at which the Mockup Studio is running.
  • On the Right, there are Global Properties and Object Properties that you can use to change global settings and properties of selected object respectively.
  • On the bottom are: Move and Rotate button that you can use to change the transform type of the selected object.
  • On the bottom right there is GET IMAGE button to generate the output of the mockup Studio.

Note:

All these elements are discussed below individually.

Interaction

Selecting

  • 1

    Yellow Selection Box

    You can select any object on the scene by simply clicking on it. A yellow box will appear around it when selected.
  • 1

    No Selection Box

    Background selection won't display any Yellow box(as the background is very large so is it's selection box), so if there is no yellow box displayed assume the background is selected.
  • 2

    No Transform Controls

    If nothing is selected no transform arrows or rotating circles are visible.
  • 2

    Orange Boxes?

    Orange boxes represent the clones of selected object

Moving

  • 1

    Rotate

    You can rotate around the scene by click and dragiing the left mouse button
  • 2

    Pan

    You can move the scene up, down, left, right click and dragiing the right mouse button
  • 3

    Zoom

    You can zoom in and zoom out by scrolling the mouse wheel.

File Menu

New (Ctrl + N)

Creates a new scene..

Anything unsaved data will be lost

Open (Ctrl + O)

Open a container with file explorer. You can select any file to load the files..

Anything unsaved data will be lost

Save (Ctrl + S)

Save the current mockup into a file. Asks for a filename if a new files or else overwrites the opended file.

Existing file name is displayed on top title that will be used to save the file.

Save As (Ctrl + Shift + S)

Save the current mockup into a new file. Asks for a filename.

Asks for a filename.

Add Menu

Copy (Ctrl + V)

Creates a copy of selected object.

Copy only looks same but doesn't share any relation to the source.

Clone (Ctrl + C)

Creates a clone of selected object.

Clone not only looks same but also shares material relation to the source.

Delete (Del)

Deletes the selected object.

Asks for a confirm dialog box.

Trifold/Bifold/etc

Creates objects as per saved presets. Submenus contain info about the type of object to be created.

These objects can be customized from the presets file.

Presets

Presets are added in order to easily change the structure of the selected brochure. Though they can be achieve by changing the object properties too.

An object(brochure) needs to be selected for the preset to work.

Global Properties

Global properties effect the scene and object within the scene.

Property Value Description
LightIntensity 0.0-0.99 Determines the brightness of the light being casted on the scene and its object. Value ranges from 0 to 1. 0 is complete darkess given ambient is also 0.
Ambient .00-0.99 Determines the brightness of the objects in the scene. If light intensity is 0 then wvery object is equally lit.
Light Position(Angle) 0-360 Angle at which the light rotates around the scene..
Light Position(Radius) 0-360 Angle at which the light rotates around the scene.
Light Position(Height) 0-360 Height at which the light rotates around the scene.
Enable Light Helper on/off Disabled by default. Enable it to see how light is getting casted (recommended for new users).
Shadow Darkness 0.0-0.99 Determines the darkness of shadows casted.
Soft shadow on/off Disabled by default (recommended) enables or disabled smooth sahodws casted on the scene.

Object Properties

Object properties effect the selected object, and it's clones, in the scene.

Property Value Description
Width 0-2000 Determines the width of the selected object
Height 0-2000 Determines the height of the selected object
First Fold Angle 0-180 Determines the fold value(in degress) of the first fold of trifold-brochure.
Second Fold Angle 0-180 Determines the fold value(in degress) of the second fold of trifold-brochure.
Gloss 0-99 Amount of gloss or shine on the object due to light.
Paper Flexiblity 0.0-1.0 Determines the flexibility of the brochure paper. 0 is fully rigid no curves, 1 is heavily curved.
Texture Bump Size 0.0-5.0 Sets the amount of bump of fold or bumps on the background.
BackImage design image Opens a Open Dialog box which allows you to select the image to be applied to back of the object. Sets outside image to brochure.
FrontImage design image Opens a Open Dialog box which allows you to select the image to be applied to front of the object. Sets inside image to brochure.
Repeat 5-200 Only available to background(Ground), sets the pattern repetition of background image.

Get Image

Finally time to see the output of the mockup created.

Just click on the button and a save dialog box will apper allowing you to save the output image in a desired location.

The output image resolution is 3000px by 2000px.

First click is usually slower as it takes time to generate soft shadow code for future use.
